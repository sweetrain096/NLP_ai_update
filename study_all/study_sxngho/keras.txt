1. Keras

   * 주요 고수준 신경망 API 가운데 하나, 파이썬(Python)으로 작성됐으며 여러 백엔드 신경망 엔진을 지원한다.

   * pip를 이용해 설치할 수 있다.

     ```python
     pip install keras
     
     
     ...
     Successfully built pyyaml
     Installing collected packages: pyyaml, keras
     Successfully installed keras-2.2.4 pyyaml-3.13
     ```

   * 사용할 때는 라이브러리를 이용하자

     ```python
     import keras
     
     Using TensorFlow backend
     ```
